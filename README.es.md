Habilidad Mago Debug para OpenClaw.

## Problema que resuelve
Utiliza IA para rastrear y eliminar mágicamente errores a través de intervenciones en tiempo de codificación.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales, listos para producción
- Enfoque de seguridad primero
- Soporte para rollback

## Uso
/debug-sorcerer

## Ejemplo
```
Analiza el código en busca de problemas de codificación.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
